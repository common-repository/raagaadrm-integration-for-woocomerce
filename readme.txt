This Wordpress plugin allows you to sell eBooks using WooCommerce 2.1+, while protecting them with RaagaaDRM.

== Description ==



By following just a few simple steps, you can immediately start selling your eBooks/Files on your WooCommerce site with industry standard DRM protection

1. First make sure you have WooCommerce installed and properly configured. This plugin will not work without WooCommerce!
2. Login to your admin interface and install the plugin
3. Enter your RaagaaDRM credentials under the WooCommerce > RaagaaDRM menu. (If there is no credentials please create by loggin in raagaatechnologies.com)
4. Create a virtual product, with the "Use RaagaaDRM" option checked in the tab.
5. Select the Book/File that was uploded in the RaagaaDRM application

Once your customers buy one of your eBooks and make payment for it, the books will be in the my account section RaagaaDRM products section where they can able to view the book


use [woo_rgdrm_prods] to list RaagaaDRM products in a short code any any page

[woo_rgdrm_prod_link prodid="xx"] 
Use this shortcode for induvidual products where xx is the product id

== Installation ==

1. Download the plugin
2. Unzip the file and upload the 'raagaadrm-for-woocomerce' directory to the `/wp-content/plugins` directory.
3. Activate the plugin through the plugins menu in WordPress.
4. Start selling!
